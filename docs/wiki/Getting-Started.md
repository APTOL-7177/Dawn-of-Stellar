## 💻 설치 및 실행

### 시스템 요구사항

| 항목 | 최소 사양 | 권장 사양 |
|------|-----------|-----------|
| **운영체제** | Windows 10, Linux, macOS | Windows 11, Ubuntu 22.04, macOS 13+ |
| **Python** | 3.10 이상 | 3.11 이상 |
| **RAM** | 512MB | 1GB 이상 |
| **저장공간** | 200MB | 500MB |
| **해상도** | 800×600 | 1920×1080 |

### 설치 방법

#### 방법 1: Git으로 설치 (권장)

```bash
# 1. Git이 설치되어 있는지 확인
git --version

# 2. 저장소 클론
git clone https://github.com/APTOL-7176/Dos.git

# 3. 디렉토리 이동
cd Dos

# 4. 가상환경 생성 (선택사항이지만 권장)
python -m venv venv

# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate

# 5. 의존성 설치
pip install -r requirements.txt

# 6. 게임 실행
python main.py
```

#### 방법 2: ZIP 파일로 설치

1. [GitHub 릴리스 페이지](https://github.com/APTOL-7176/Dos/releases)에서 최신 버전 다운로드
2. ZIP 파일 압축 해제
3. 터미널/명령 프롬프트로 압축 해제한 폴더로 이동
4. `pip install -r requirements.txt` 실행
5. `python main.py` 실행

### 실행 옵션

```bash
# 기본 실행
python main.py

# 개발 모드 (모든 직업 잠금 해제)
python main.py --dev

# 디버그 모드 (상세 로그 출력)
python main.py --debug

# 로그 레벨 설정
python main.py --log=DEBUG    # 가장 상세
python main.py --log=INFO     # 일반 정보
python main.py --log=WARNING  # 경고만
python main.py --log=ERROR    # 에러만

# 풀스크린 모드
python main.py --fullscreen

# 해상도 설정
python main.py --resolution=1920x1080
```

### 문제 해결

**문제: `ModuleNotFoundError: No module named 'tcod'`**
```bash
# 해결: TCOD 재설치
pip install --upgrade tcod
```

**문제: `ImportError: DLL load failed`** (Windows)
```bash
# 해결: Visual C++ 재배포 패키지 설치
# https://aka.ms/vs/17/release/vc_redist.x64.exe 다운로드 및 설치
```

**문제: 게임이 실행되지 않음**
```bash
# 1. Python 버전 확인
python --version  # 3.10 이상이어야 함

# 2. 의존성 재설치
pip uninstall -r requirements.txt -y
pip install -r requirements.txt

# 3. 로그 확인
python main.py --debug --log=DEBUG
```

## ⚔️ 전투 기초

### ATB 시스템

```
ATB 게이지: [████████░░] 800 / 2000
             행동 가능 →  1000
```

- **ATB 게이지**는 0에서 시작하여 매 프레임 증가
- **1000**에 도달하면 행동 가능
- **속도** 스탯이 높을수록 빠르게 증가

### BRV 시스템

```
캐릭터 BRV: 1200 / 1500
적 BRV:      800 / 1000
```

**기본 전투 흐름**:
1. **BRV 공격** (Q키) - 적의 BRV를 깎고 내 BRV 축적
2. **BRV 축적** - 1500까지 축적 가능
3. **HP 공격** (E키) - 축적된 BRV로 실제 HP 데미지
4. **BRV 초기화** - HP 공격 후 BRV는 기본값으로

### BREAK 시스템

```
적 BRV: 300 → 0  [BREAK!]
```

**BREAK 조건**: 적의 BRV를 정확히 0으로 만들 때

**BREAK 효과**:
- 🎯 다음 HP 공격 데미지 **+50%**
- 💫 적 **1턴 스턴** (행동 불가)
- ⚡ 자신의 BRV **+20%**

💡 **전투 팁**: BREAK를 노려 보너스 데미지를 획득하세요!

### 첫 전투 예시

**시나리오**: 고블린 1마리 조우

```
[턴 1] ATB 1000 도달
→ BRV 공격 (Q키) 선택
→ 고블린 BRV 1000 → 700 (-300)
→ 내 BRV 1000 → 1300 (+300)

[턴 2] ATB 1000 도달
→ BRV 공격 (Q키) 선택
→ 고블린 BRV 700 → 400 (-300)
→ 내 BRV 1300 → 1500 (최대치)

[턴 3] 고블린 공격
→ 내 BRV 1500 → 1300 (-200)

[턴 4] ATB 1000 도달
→ HP 공격 (E키) 선택
→ 고블린 HP 100 → 34 (-66 HP)
→ 내 BRV 1500 → 1000 (초기화)

[턴 5] ATB 1000 도달
→ BRV 공격 (Q키) 선택
→ 고블린 BRV 400 → 0 (BREAK!)
→ 내 BRV 1000 → 1400

[턴 6] ATB 1000 도달
→ HP 공격 (E키, BREAK 보너스!)
→ 고블린 HP 34 → 0 (사망!)

승리!
```

## 🎭 초보자 추천 직업

### 1위: 전사 (Warrior) ⭐⭐⭐⭐⭐

**난이도**: ★☆☆☆☆ (매우 쉬움)

**장점**:
- ❤️ 높은 HP (210)와 방어력 (28)
- 🛡️ 다양한 스탠스로 상황 대응
- ⚔️ 물리 공격과 방어 모두 우수
- 🔄 실수해도 살아남기 쉬움

**기본 전략**:
```
1. 전투 시작: "공격 자세" 스탠스
2. BRV 공격 2회 → HP 공격 1회
3. HP 낮으면: "방어 자세"로 전환
4. 강한 적: "균형 자세"로 안정화
```

**추천 스킬**:
- **강타** (BRV 공격, 배율 2.0)
- **십자 베기** (HP 공격)
- **방패 밀쳐내기** (적 밀치기)

**초반 패시브**:
- HP 증폭 (+20% HP)
- 신속 (+12% 속도)
- 자동 재생 (턴당 HP 1% 회복)

**난이도**: ★☆☆☆☆ (쉬움)

- ❤️ 강력한 회복 스킬

**기본 전략**:
```
1. HP 50% 이하: "케어" 사용
2. 전투 전: "프로텍트" 버프
3. BRV 공격: "신성한 벌"

```

**추천 스킬**:
- **케어** (HP 회복 100)
- **프로텍트** (물리 피해 -30%)
- **신성한 벌** (BRV 공격, 신성 속성)

**추천 스킬**:
- **정밀 사격** (조준 5스택 소모, 데미지 3배)
- **연사** (BRV 공격 3회)
- **약점 노출** (방어력 -30%)

## 🏰 첫 던전 공략

### 던전 1층 (튜토리얼)

**목표**: 계단 찾아서 2층으로 이동

**적 종류**:
- **슬라임** (HP: 50, 약함)
- **고블린** (HP: 100, 기본)

**아이템**:
- 🍖 고기 (HP +20)
- 💧 물 (MP +10)
- 💰 골드 (10~30)

**팁**:
1. **모든 방 탐험** - 아이템과 골드 획득
2. **전투는 선택적** - 피할 수 있으면 피하세요
3. **HP 관리** - HP 50% 이하면 회복 아이템 사용
4. **계단 찾기** - 보통 맵 끝쪽에 위치

### 던전 2층 (기본)

**목표**: 보스 처치 후 3층으로

**적 종류**:
- **고블린 전사** (HP: 150, 방어력 높음)
- **고블린 주술사** (HP: 100, 마법 공격)

**보스**: **고블린 대장** (HP: 300)

**보스 공략**:
```
[페이즈 1] HP 100% ~ 50%
- 기본 공격만 사용
- BRV 축적 후 HP 공격

[페이즈 2] HP 50% ~ 0%
- "분노" 버프 사용 (공격력 +30%)
- 강력한 공격 주의!
- 방어 자세로 대응
```

**팁**:
1. **보스 전 준비** - HP/MP 최대로 회복
2. **요리 사용** - 구운 고기 (+40 HP)
3. **BREAK 노리기** - 보너스 데미지 활용
4. **패턴 파악** - 2턴마다 강공격

## ❌ 자주 하는 실수

### 실수 1: BRV를 너무 많이 쌓음

❌ **잘못된 플레이**:
```
내 BRV: 1500 (최대치)
→ BRV 공격 계속 시도
→ BRV 손실! (넘치는 BRV 사라짐)
```

✅ **올바른 플레이**:
```
내 BRV: 1400
→ HP 공격으로 데미지
→ BRV 초기화 후 다시 축적
```

💡 **팁**: BRV 최대치(1500)의 80% 정도면 HP 공격하세요!

---

### 실수 2: MP 낭비

❌ **잘못된 플레이**:
```
약한 적 (슬라임)
→ 궁극기 사용 (MP 50 소모)
→ MP 부족으로 보스전 고생
```

✅ **올바른 플레이**:
```
약한 적: 기본 공격 (무료)
강한 적/보스: 스킬 사용
```

💡 **팁**: MP는 보스전을 위해 아껴두세요!

---

### 실수 3: 패시브 스킬 무시

❌ **잘못된 플레이**:
```
레벨업 → 별조각 획득
→ 그냥 방치
→ 약한 캐릭터
```

✅ **올바른 플레이**:
```
레벨업 → 별조각 획득
→ 패시브 스킬 구매
→ HP +20%, 속도 +12% 등
```

💡 **팁**: 초반 추천 패시브는 **HP 증폭**, **신속**, **자동 재생**!

---

### 실수 4: 요리 시스템 무시

❌ **잘못된 플레이**:
```
보스전 직전
→ 요리 안 함
→ 약한 상태로 전투
```

✅ **올바른 플레이**:
```
보스전 직전 휴식
→ 구운 고기 요리 (HP +40)
→ 버프 받고 전투
```

💡 **팁**: 요리는 전투 전 필수! MP 15로 큰 효과!

---

### 실수 5: 장비 업그레이드 무시

❌ **잘못된 플레이**:
```
새 장비 획득
→ 그냥 인벤토리 보관
→ 약한 장비로 전투
```

✅ **올바른 플레이**:
```
새 장비 획득
→ 즉시 장착 (I키 → 장비)
→ 스탯 증가
```

💡 **팁**: 드랍된 장비는 바로 확인하고 좋으면 장착!

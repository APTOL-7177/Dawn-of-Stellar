## 🎮 게임 기본

### Q: Dawn of Stellar는 어떤 게임인가요?

**A:** Final Fantasy 스타일의 ATB + BRV 전투 시스템을 가진 로그라이크 RPG입니다. 33개의 고유한 직업, 195개 이상의 스킬, 절차적으로 생성되는 던전을 특징으로 합니다.

---

### Q: 무료인가요?

**A:** 네! 완전 무료이며 MIT 라이선스로 오픈소스입니다.

---

### Q: 한국어를 지원하나요?

**A:** 네! 모든 UI, 텍스트, 튜토리얼이 완전히 한국어로 번역되어 있습니다.

---

### Q: 멀티플레이어가 가능한가요?

**A:** 네! 버전 6.1.0부터 멀티플레이어를 지원합니다. 최대 4인까지 협동 플레이가 가능합니다.

---

### Q: 게임을 저장할 수 있나요?

**A:** 네! 던전 중간에 저장 가능하며, 여러 세이브 슬롯을 지원합니다. 휴식 시점에 자동 저장됩니다.

---

## 💻 설치 및 실행

### Q: Python 버전이 3.10 미만인데 실행할 수 있나요?

**A:** 아니요. Python 3.10 이상이 필수입니다. `python --version`으로 버전을 확인하고, 필요시 업그레이드하세요.

```bash
# Python 버전 확인
python --version

# Python 3.10 이상이 아니라면 업그레이드
# https://www.python.org/downloads/
```

---

### Q: `ModuleNotFoundError: No module named 'tcod'` 에러가 발생합니다.

**A:** TCOD 라이브러리가 설치되지 않았습니다.

```bash
# 해결방법
pip install tcod

# 또는 전체 의존성 재설치
pip install -r requirements.txt
```

---

### Q: 게임이 느리게 실행됩니다.

**A:** 다음을 시도해보세요:

1. **디버그 모드 끄기**
   ```bash
   # 디버그 없이 실행
   python main.py
   ```

2. **해상도 낮추기**
   ```bash
   python main.py --resolution=1280x720
   ```

3. **Python 최적화 모드**
   ```bash
   python -O main.py
   ```

---

### Q: Windows에서 `DLL load failed` 에러가 발생합니다.

**A:** Visual C++ 재배포 패키지가 필요합니다.

1. [Microsoft Visual C++ 재배포 패키지](https://aka.ms/vs/17/release/vc_redist.x64.exe) 다운로드
2. 설치 후 재부팅
3. 게임 재실행

## ⚔️ 전투 시스템

### Q: ATB가 무엇인가요?

**A:** **Active Time Battle** (실시간 전투) 시스템입니다.

- ATB 게이지가 0에서 시작하여 매 프레임 증가
- 1000 도달 시 행동 가능
- 속도 스탯이 높을수록 빠르게 증가

자세한 내용은 [전투 시스템 가이드](../combat-system.md)를 참조하세요.

---

### Q: BRV가 무엇인가요?

**A:** **Brave** (용기) 포인트로, 실제 HP 데미지를 주기 전 축적하는 중간 단계입니다.

```
BRV 공격 → BRV 축적 → HP 공격 → 실제 HP 데미지
```

- **BRV 공격**: HP 데미지 없이 BRV만 축적
- **HP 공격**: 축적된 BRV로 HP 데미지
- **BREAK**: 적 BRV를 0으로 만들면 보너스

---

### Q: BREAK는 언제 발동하나요?

**A:** 적의 BRV를 **정확히 0**으로 만들 때 발동합니다.

**BREAK 효과**:
- 다음 HP 공격 데미지 +50%
- 적 1턴 스턴
- 자신의 BRV +20%

💡 **팁**: BREAK를 노려 보너스 데미지를 획득하세요!

---

### Q: 크리티컬 확률은 어떻게 계산되나요?

**A:**
```
크리티컬 확률 = 기본 10% + (행운 / 100)
```

**예시**:
- 행운 0: 10%
- 행운 10: 20%
- 행운 20: 30%

**크리티컬 데미지**: 1.5배 (패시브로 2.0배 가능)

---

### Q: 속성 시스템은 어떻게 작동하나요?

**A:** 약점 공격 시 **1.5배** 데미지, 저항 시 **0.5배** 데미지입니다.

**속성 상성**:
- 화염 → 빙결 (약점)
- 빙결 → 번개 (약점)
- 번개 → 대지 (약점)
- 대지 → 바람 (약점)
- 바람 → 화염 (약점)
- 신성 ↔ 암흑 (상호 약점)

## 🎭 직업 및 스킬

### Q: 초보자에게 추천하는 직업은?

**A:**
1. **전사** - 높은 HP, 균형잡힌 스탯
2. **성직자** - 회복 특화, 안전한 플레이
3. **궁수** - 원거리 공격, 안전한 딜링

자세한 내용은 [시작 가이드](Getting-Started.md)를 참조하세요.

---

### Q: 직업을 바꿀 수 있나요?

**A:** 게임 중에는 변경할 수 없습니다. 새 게임을 시작하거나 세이브 파일을 선택할 때 직업을 선택할 수 있습니다.

---

### Q: 직업은 어떻게 해금하나요?

**A:** **별조각**을 사용하여 해금합니다.

**해금 비용**:
- 기본 (5개): 0 별조각 (초기 해금)
- 일반 (8개): 100 별조각
- 고급 (10개): 200 별조각
- 희귀 (7개): 400 별조각
- 전설 (3개): 800 별조각

자세한 내용은 [메타 진행 가이드](Meta-Progression.md)를 참조하세요.

---

### Q: 스킬은 어떻게 배우나요?

**A:** 각 직업은 **6개의 고유 스킬**을 자동으로 가지고 있습니다. 레벨업이나 별도 학습이 필요 없습니다.

---

### Q: MP가 부족합니다. 어떻게 관리하나요?

**A:**
1. **기본 공격 활용** - MP 소모 없음
2. **MP 포션 휴대** - 긴급 시 사용
3. **패시브 "마나 회복"** - 턴당 MP 2% 회복
4. **요리 활용** - 마나 커피 (MP +100)
5. **휴식** - 안전 지대에서 MP 회복

---

## 🎒 아이템 및 장비

### Q: 인벤토리가 가득 찼습니다.

**A:**
1. **불필요한 아이템 버리기** - I키 → 아이템 선택 → D키
2. **인벤토리 확장** - 별조각으로 영구 업그레이드 (100 별조각)
3. **요리로 소비** - 재료를 요리로 변환

---

### Q: 장비는 어떻게 장착하나요?

**A:**
```
1. I키 (인벤토리)
2. 장비 탭 선택
3. 장착할 장비 선택
4. Enter 또는 E키
```

---

### Q: 장비 등급은?

**A:**
- **일반** (회색) - 기본 효과
- **고급** (녹색) - 효과 +20%
- **희귀** (파랑) - 효과 +50%
- **영웅** (보라) - 효과 +100%
- **전설** (금색) - 효과 +200% + 특수 효과

---

### Q: 요리는 어떻게 하나요?

**A:**
```
1. 안전 지대에서 휴식 (R키)
2. "요리하기" 선택
3. 레시피 선택
4. MP 15 소모
5. 요리 완성!
```

52개 레시피는 [요리 가이드](../cooking.md)를 참조하세요.

---

## ⭐ 메타 진행

### Q: 별조각은 어떻게 얻나요?

**A:**
- **레벨업**: 10~30 별조각
- **보스 처치**: 30~100 별조각
- **층 클리어**: 5~50 별조각
- **업적 달성**: 50~500 별조각

자세한 내용은 [메타 진행 가이드](Meta-Progression.md)를 참조하세요.

---

### Q: 패시브 스킬은 어떻게 구매하나요?

**A:**
```
1. 메인 메뉴 → "메타 진행"
2. "패시브 스킬" 선택
3. 구매할 스킬 선택
4. 별조각 소모
```

---

### Q: 패시브 코스트가 부족합니다.

**A:** **레벨업**으로 코스트를 증가시킬 수 있습니다.

```
레벨 1:  5 코스트
레벨 10: 15 코스트
레벨 20: 30 코스트
최대:    50 코스트 (레벨 50)
```

---

### Q: 어떤 패시브를 먼저 구매해야 하나요?

**A:** 초보자 추천:
1. **HP 증폭** (3 코스트) - HP +20%
2. **신속** (2 코스트) - 속도 +12%
3. **자동 재생** (3 코스트) - 턴당 HP 1% 회복
4. **마나 회복** (3 코스트) - 턴당 MP 2% 회복

---

## 🔧 기술적 문제

### Q: 저장 파일이 손상되었습니다.

**A:**
```
1. saves/ 폴더로 이동
2. 백업 파일 확인 (.bak)
3. .bak를 원본으로 복사
4. 게임 재실행
```

---

### Q: 게임이 충돌합니다.

**A:**
1. **로그 확인**
   ```bash
   logs/game_latest.log 파일 확인
   ```

2. **디버그 모드로 실행**
   ```bash
   python main.py --debug --log=DEBUG
   ```

3. **GitHub 이슈 제출**
   - 로그 파일 첨부
   - 재현 방법 설명

---

### Q: 화면이 깨집니다.

**A:**
1. **터미널 인코딩 확인**
   ```bash
   # Windows
   chcp 65001

   # 그 후 게임 실행
   python main.py
   ```

2. **폰트 변경** - `config.yaml`에서 폰트 설정

---

### Q: 사운드가 재생되지 않습니다.

**A:**
1. **오디오 파일 확인**
   ```bash
   # assets/audio/ 폴더 확인
   ```

2. **볼륨 설정**
   - O키 → 옵션 → 볼륨

3. **오디오 라이브러리 재설치**
   ```bash
   pip install --upgrade pygame
   ```

## 💡 게임 플레이 팁

### Q: 첫 던전에서 계속 죽습니다.

**A:**
1. **난이도 낮추기** - "평온" 선택
2. **전사 직업 사용** - 초보자 친화적
3. **요리 활용** - 전투 전 버프
4. **휴식 자주** - HP/MP 회복
5. **도망치기** - 강한 적은 피하기

---

### Q: 보스를 이길 수 없습니다.

**A:**
1. **보스 전 준비**
   - HP/MP 최대로 회복
   - 요리로 버프 (구운 고기, 스테이크)
   - 포션 충분히 준비

2. **보스 패턴 파악**
   - 페이즈별 패턴 학습
   - BREAK 노리기
   - 위험 신호 주의

3. **최적 빌드**
   - 보스 특화 패시브
   - 화력 집중
   - 생존 우선

자세한 내용은 [보스 공략 가이드](Boss-Strategies.md)를 참조하세요.

---

### Q: 골드가 부족합니다.

**A:**
1. **해적 직업** - 골드 획득 +40%
2. **영구 업그레이드** - "골드 증가" 구매
3. **상인 이벤트** - 아이템 판매
4. **보스 처치** - 높은 골드 보상

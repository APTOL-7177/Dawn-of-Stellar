## 🌟 패시브 시스템 개요

### 기본 개념

**패시브 스킬**은 항상 활성화된 영구 강화 효과입니다.

- 별조각으로 해금
- 코스트 소모하여 장착
- 레벨업으로 코스트 증가
- 자유롭게 조합 가능

### 코스트 시스템

**총 코스트**는 레벨에 따라 증가:

```
레벨 1:   5 코스트
레벨 5:   10 코스트
레벨 10:  15 코스트
레벨 15:  20 코스트
레벨 20:  30 코스트
레벨 30:  40 코스트
레벨 50:  50 코스트 (최대)
```

## 📈 최적화 가이드

### 레벨별 추천 패시브

**레벨 1-10 (10 코스트)**:
```
- HP 증폭 (3)
- 신속 (2)
- 자동 재생 (3)

→ 기본 생존 세팅
```

**레벨 11-20 (25 코스트)**:
```
+ 힘의 축복 (4)
+ 치명타 강화 (3)
+ 마나 회복 (3)
+ 선제공격 (4)

→ 화력 + 효율
```

**레벨 21-30 (40 코스트)**:
```
+ 치명타 위력 (4)
+ BREAK 보너스 (5)
+ 불사조의 가호 (6)

→ 완성형 빌드
```

### 코스트 효율 분석

**가성비 최고**:
1. **신속** (2 코스트) - 속도 +12%
2. **행운의 별** (2 코스트) - 행운 +10
3. **HP 증폭** (3 코스트) - HP +20%
4. **치명타 강화** (3 코스트) - 크리티컬 +15%

**가성비 중간**:
- 대부분의 4~5 코스트 패시브

**가성비 낮음**:
- **신의 축복** (12 코스트) - 효과는 좋지만 비쌈

## ⚔️ 전투 팁

### 💡 BRV 관리의 황금률

**80/20 규칙**:
- BRV가 최대치의 **80%**에 도달하면 HP 공격
- 더 쌓으면 BREAK 당할 위험

```
최대 BRV: 1500
→ 1200 도달 시 HP 공격
→ 안전하게 데미지 + BRV 초기화
```

---

### 💡 BREAK 정확히 노리기

**계산법**:
```
적 BRV: 300
내 BRV 공격력: 350
→ 300 대미지로 조정 필요

방법: 약한 스킬 사용 or 디버프 적용
```

**꿀팁**: 정확히 0으로 만들 자신 없으면, **조금 적게** 깎는 것이 안전!

---

### 💡 ATB 조작 테크닉

**헤이스트 + 슬로우 조합**:
```
1. 아군에게 헤이스트 (ATB +50%)
2. 적에게 슬로우 (ATB -50%)
→ 행동 빈도 3배 차이!
```

**선제공격 패시브**:
- 전투 시작 ATB +50%
- 보스전에서 첫 공격 확보

---

### 💡 상태 효과 스택

**공격력 증가 중첩**:
```
버프 1: 공격력 +20%
버프 2: 공격력 +20%
버프 3: 공격력 +20%
→ 총 +60% (최대 3스택)
```

**독 중첩**:
```
독 1: 3%/턴
독 2: 3%/턴
...
독 5: 3%/턴
→ 총 15%/턴 (최대 5스택)
```

---

### 💡 MP 절약 전략

**무료 전투 패턴**:
```
1. 기본 공격 (Q키) - MP 0
2. 기본 공격 (Q키) - MP 0
3. HP 공격 (E키) - MP 0
→ 약한 적은 MP 사용 안함!
```

**마나 회복 타이밍**:
- 전투 후 휴식 (R키) - 무료 회복
- 안전 지대 도달 시 최대 회복
- MP 포션은 보스전 전용

---

### 💡 크리티컬 확정 방법

**100% 크리티컬**:
1. **저격수** - 장거리 지배 패시브
2. **사무라이** - 발도 기술
3. **암살자** - 은신 후 암살
4. **행운 90** - 10% + 90% = 100%

---

### 💡 상처 시스템 활용

**상처 메커니즘**:
```
HP 데미지 100
→ 실제 HP -100
→ 상처 +25 (25%)

최대 상처: 최대 HP의 50%
```

**상처 관리**:
- 휴식으로 상처 50% 회복
- 아이템 사용 시 회복 효율 1.5배
- 상처 많으면 최대 HP 감소

💡 **팁**: 보스전 전 반드시 휴식으로 상처 제거!

## 🗺️ 탐험 팁

### 💡 던전 생성 패턴

**BSP 알고리즘 이해**:
```
┌─────────┬─────────┐
│  방1    │  방2    │
├─────────┼─────────┤
│  방3    │  방4    │
└─────────┴─────────┘

계단은 보통 맵 끝(모서리)에 위치
```

**효율적 탐험**:
1. 시작 지점에서 벽 따라 이동
2. 모든 방 체크
3. 막다른 방은 마지막에

---

### 💡 안전 지대 찾기

**안전 지대 특징**:
- 적 스폰 없음
- HP/MP 자동 회복
- 요리 및 휴식 가능
- 저장 가능

**출현 패턴**:
- 5층마다 1개 보장
- 큰 방에 주로 위치
- "🕯️" 아이콘으로 표시

---

### 💡 숨겨진 아이템

**벽 체크**:
```
벽에 가까이 가면 숨겨진 문/상자 발견
"!" 표시 확인
Enter로 상호작용
```

**무당의 시야**:
- 시야 2배
- 숨겨진 것 자동 발견
- 미니맵에 모든 것 표시

---

### 💡 적 회피 테크닉

**적 시야 범위**:
```
시야: 5칸
청각: 3칸 (소음)

회피 방법:
1. 적 시야 밖으로 우회
2. 소음 최소화 (천천히 이동)
3. 도적의 은신 사용
```

## 📊 효율성 팁

### 💡 레벨링 최적화

**경험치 효율**:
```
평온 난이도: 경험치 120%
모든 전투 참여
경험치 포션 사용
파티 플레이 (+20%)

→ 레벨 20까지 2시간
```

**빠른 레벨링 루트**:
1. 적 밀집 지역 찾기
2. 광역 스킬 사용
3. 보스 반복 처치
4. 경험치 버프 요리

---

### 💡 골드 파밍

**골드 효율 직업**:
1. **해적** - +40% 골드
2. **도적** - 아이템 훔치기
3. **무당** - 숨겨진 보물

**골드 파밍 루트**:
```
1. 해적 선택
2. 영구 업그레이드 "골드 증가" (+10%)
3. 보스 반복 처치
4. 상인 이벤트 활용

→ 1시간에 10,000 골드
```

---

### 💡 아이템 드랍률 증가

**드랍 증가 방법**:
1. 난이도 상승 (도전: +20%, 악몽: +50%, 지옥: +100%)
2. 영구 업그레이드 (+10%)
3. 행운 스탯 증가
4. 해적/도적 직업

**희귀 아이템 파밍**:
```
지옥 난이도
+ 해적 직업
+ 영구 업그레이드
+ 행운 30
→ 드랍률 약 200%
```

---

### 💡 별조각 파밍

**최고 효율 방법**:
```
보스 러시 전략:
1. 보통 난이도
2. 검성 or 광전사
3. 보스까지 직행
4. 보스만 처치
5. 다음 층 반복

→ 시간당 200 별조각
```

## 🎭 직업별 팁

### 전사
- **스탠스 변경**은 무료 - 자주 활용
- 보스전: 광전사 자세로 최대 화력
- 위험 시: 수호자 자세 (방어력 +50%)

### 검성
- **검기 5스택 유지** - 항상 준비
- 크리티컬 패시브 필수
- 검기 폭발은 보스전 전용

### 아크메이지
- **원소 균형** 유지 - 3속성 고루
- MP 관리 필수 - 마나 회복 패시브
- 약점 속성 파악 후 집중

### 시간술사
- **시간 역행**은 보스전 필수
- 헤이스트 + 슬로우 조합 강력
- 파티에서 최고의 서포터

### 암살자
- **은신 → 암살** 콤보
- 배후 공격 +50% 데미지
- 약한 적 우선 처치 (암살술)

### 바드
- **전투 시작 즉시 버프**
- 파티 규모가 클수록 강력
- MP 관리 (앙코르 패시브)

### 광전사
- **HP 30%~40% 유지** - 최적 DPS
- 포션 항상 휴대
- 불멸의 분노 (부활) 활용

### 해적
- **골드 파밍 최적화**
- 탐욕 패시브 (골드 = 스탯)
- 비열한 전투 (랜덤 디버프)

## 🔍 숨겨진 메커니즘

### 💡 히든 패시브 효과

**일부 패시브는 설명에 없는 효과**:

**HP 증폭**:
- 설명: HP +20%
- **숨겨진**: 상처 회복 속도 +10%

**신속**:
- 설명: 속도 +12%
- **숨겨진**: 회피 확률 +3%

**자동 재생**:
- 설명: 턴당 HP 1% 회복
- **숨겨진**: 상처 자동 치료

---

### 💡 요리 품질 비밀

**품질 결정 요소**:
```
기본 확률:
Poor: 20%
Normal: 50%
Good: 25%
Excellent: 5%

+ 손재주 스탯 (+1% per 10 손재주)
+ 연금술사 직업 (+20%)
+ 요리 도구 장비 (+15%)

→ Excellent 최대 약 40%
```

---

### 💡 보스 숨겨진 패턴

**대부분의 보스는 HP 50% 이하에서 강화**:
- 공격력 +30%
- 속도 +20%
- 새로운 스킬 사용

**대응**:
- HP 51%에서 대기
- 버프 준비
- HP 50% 돌파 시 즉시 폭딜

---

### 💡 층별 보너스

**10층마다 특별 보상**:
```
10층: 희귀 아이템
20층: 전설 장비
30층: 대량 별조각 (200개)
50층: 특수 직업 해금
100층: 게임 클리어 보상
```

## 📋 치트 시트

### 초보자 체크리스트

게임 시작 시:
- [ ] 난이도 "평온" 선택
- [ ] 전사/성직자/궁수 선택
- [ ] 패시브 3개 구매 (HP 증폭, 신속, 자동 재생)
- [ ] 조작법 숙지
- [ ] 첫 던전 진입

전투 시:
- [ ] BRV 80% 도달 시 HP 공격
- [ ] BREAK 노리기
- [ ] MP 절약 (약한 적은 기본 공격)
- [ ] HP 50% 이하 시 회복

탐험 시:
- [ ] 모든 방 탐험
- [ ] 숨겨진 아이템 체크
- [ ] 안전 지대에서 휴식
- [ ] 보스 전 준비 (요리, 포션)

## 💎 프로 플레이어 비밀

### 최고 효율 빌드

**속도 런 빌드**:
```
직업: 도적
패시브:
- 신속 (속도 +12%)
- 선제공격 (ATB +50%)
- 완벽한 회피 (회피 +25%)
- 치명타 강화/위력

전략: 회피 + 속도로 적 공격 받지 않고 빠른 처치
```

**불사 빌드**:
```
직업: 성기사
패시브:
- HP 증폭 (HP +20%)
- 강인함 (피해 -10%)
- 자동 재생 (HP 1%/턴)
- 불사조의 가호 (부활)
- 불멸의 영혼 (무적 3턴)

전략: 절대 죽지 않는 탱커
```

**폭딜 빌드**:
```
직업: 광전사
패시브:
- 치명타 강화 (크리티컬 +15%)
- 치명타 위력 (2.0배)
- 힘의 축복 (공격력 +15%)
- 궁극의 힘 (모든 스탯 +10%)

전략: 저 HP 유지 + 분노 10스택 + 크리티컬 = 원샷
```

### 100층 클리어 팁

**준비**:
- 레벨 50 이상
- 패시브 30개 이상
- 전설 장비 풀세팅
- 요리 대량 비축

**직업 추천**:
1. 시간술사 (시간 역행)
2. 차원술사 (평행우주)
3. 성기사 (생존)

**전략**:
- 50층마다 휴식 필수
- 보스 패턴 완벽 숙지
- 포션 아끼지 말기
- 시간 역행/평행우주 활용

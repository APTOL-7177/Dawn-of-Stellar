## ⚔️ 전투 시스템 개요

Dawn of Stellar는 **Final Fantasy VII의 ATB**와 **Dissidia의 BRV 시스템**을 결합한 독창적인 전투 시스템을 사용합니다.

### 핵심 메커니즘

```
ATB 게이지 충전 → 행동 선택 → BRV 축적 → HP 공격 → 승리!
```

### 전투 흐름

1. **전투 시작**: 모든 전투원 ATB 0에서 시작
2. **ATB 증가**: 매 프레임마다 속도에 비례하여 증가
3. **행동 턴**: ATB ≥ 1000일 때 행동 가능
4. **스킬 실행**: BRV 공격, HP 공격, 스킬, 아이템 사용
5. **승리/패배 판정**: 모든 적 또는 아군 전멸 시 종료

## ⏱️ ATB (Active Time Battle)

### 기본 설정

| 항목 | 값 | 설명 |
|------|-----|------|
| 최대 게이지 | 2000 | ATB 게이지 최대값 |
| 행동 임계값 | 1000 | 행동 가능 게이지 |
| 기본 증가율 | 50/프레임 | 평균 속도 기준 |
| 최대 오버플로우 | 2000 | 게이지 상한 |

### ATB 증가 공식

```python
ATB 증가량 = (캐릭터 속도 / 평균 속도) × 50
```

**예시:**
- 캐릭터 속도 60, 평균 속도 50
- ATB 증가 = (60 / 50) × 50 = **60/프레임**
- 행동까지 시간 = 1000 / 60 = **약 16.7프레임**

### 속도의 중요성

| 속도 | ATB 증가 | 행동 주기 | 평가 |
|------|----------|-----------|------|
| 40 | 40/프레임 | 25 프레임 | 느림 |
| 50 | 50/프레임 | 20 프레임 | 보통 |
| 60 | 60/프레임 | ~17 프레임 | 빠름 |
| 80 | 80/프레임 | 12.5 프레임 | 매우 빠름 |
| 100 | 100/프레임 | 10 프레임 | 초고속 |

### ATB 관리 전략

**1. 속도 스탯 투자**
- 속도 +10 → 행동 빈도 +20%
- 패시브 "신속" (속도 +12%) 추천

**2. 헤이스트 버프**
- ATB 증가율 +50%
- 시간술사의 "헤이스트" 스킬

**3. 선제공격 패시브**
- 전투 시작 시 ATB +50%
- 코스트 4, 해금 400 별조각

## 💪 BRV (Brave) 시스템

### BRV란?

**BRV**는 실제 HP 데미지를 주기 전 축적하는 중간 단계의 "용기" 포인트입니다.

```
BRV 공격 (적의 BRV를 깎고 내 BRV 축적)
     ↓
BRV 축적 (공격력만큼 BRV 포인트 획득)
     ↓
HP 공격 (축적된 BRV로 HP 데미지)
```

### BRV 기본 설정

| 항목 | 값 | 설명 |
|------|-----|------|
| 기본 BRV | 1000 | 전투 시작 BRV |
| 최대 BRV | 기본 × 1.5 | BRV 상한 (1500) |
| BREAK 임계값 | 0 | 적 BRV가 0이 되면 BREAK |

### BRV 공격 vs HP 공격

| 구분 | BRV 공격 | HP 공격 |
|------|----------|---------|
| **목적** | BRV 축적 | HP 데미지 |
| **HP 데미지** | 없음 | 축적된 BRV만큼 |
| **BRV 소비** | 없음 | 전부 소비 (0으로 초기화) |
| **위험도** | 낮음 | 높음 (BRV 초기화) |
| **사용 시점** | BRV 부족할 때 | BRV 충분할 때 |

### BREAK 시스템

**BREAK 조건**: 적의 BRV를 정확히 0으로 만들 때

**BREAK 효과**:
1. **보너스 데미지**: 다음 HP 공격 +50%
2. **스턴**: 적 1턴 행동 불가
3. **BRV 보너스**: 자신의 BRV +20%

**BREAK 전략**:
```
1단계: 적 BRV 확인 (예: 300)
2단계: BRV 공격으로 정확히 300 깎기
3단계: BREAK! 보너스 획득
4단계: HP 공격으로 마무리 (+50% 데미지)
```

### BRV 관리 전략

**초보자 전략: 안전 플레이**
```
BRV 공격 x3 → HP 공격 x1 → 반복
```
- BRV를 충분히 쌓은 후 공격
- 안정적이지만 느림

**중급자 전략: BREAK 노리기**
```
BRV 공격 x2 → BREAK 유도 → HP 공격 (보너스)
```
- 적 BRV를 0으로 만들어 보너스 획득
- 위험도 중간, 효율 높음

**고급자 전략: 최적화**
```
BRV 공격 (적 BRV 낮을 때) → HP 공격 (적 BRV 높을 때)
```
- 상황에 따라 유연하게 대응
- 최고 효율, 높은 난이도

## 🎯 데미지 계산

### BRV 데미지 공식

```python
BRV 데미지 = (공격력 / 방어력) × 스킬배율 × 75.0 × 랜덤(0.9~1.1) × 크리티컬
```

**배율 설명**:
- **스킬배율**: 스킬마다 다름 (1.0~3.0)
- **BRV 배율**: 75.0 (고정)
- **랜덤**: 0.9~1.1 (±10% 변동)
- **크리티컬**: 1.5배 (발동 시)

**예시 계산**:
```
전사 (공격력 60) vs 고블린 (방어력 30)
스킬배율: 2.0 (강타)
크리티컬: 미발동

BRV 데미지 = (60 / 30) × 2.0 × 75.0 × 1.0 × 1.0
           = 2 × 2.0 × 75.0
           = 300 BRV
```

### HP 데미지 공식

```python
HP 데미지 = BRV 포인트 × 0.25 × 스탯보정 × BREAK보너스
상처 데미지 = HP 데미지 × 0.25
```

**배율 설명**:
- **HP 배율**: 0.25 (BRV의 1/4만 HP 데미지)
- **스탯보정**: 공격력 / (방어력 + 1)
- **BREAK 보너스**: 1.5배 (BREAK 시)
- **상처 데미지**: HP 데미지의 25%

**예시 계산**:
```
축적 BRV: 1000
스탯보정: 60 / 31 = 1.935
BREAK: 발동 (1.5배)

HP 데미지 = 1000 × 0.25 × 1.935 × 1.5
          = 1000 × 0.25 × 2.9025
          = 약 726 HP

상처 데미지 = 726 × 0.25 = 181 상처
```

### 크리티컬 시스템

**크리티컬 확률**:
```python
크리티컬 확률 = 기본 10% + (행운 / 100)
```

**예시**:
- 행운 0: 10% 확률
- 행운 10: 20% 확률
- 행운 20: 30% 확률

**크리티컬 효과**:
- 데미지 1.5배
- BRV 공격, HP 공격 모두 적용

**크리티컬 강화 방법**:
1. **행운 스탯 투자**
2. **패시브 "치명타 강화"** (코스트 3, +15%)
3. **장비 효과** (크리티컬 확률 +X%)

### 속성 시스템

| 속성 | 약점 | 저항 |
|------|------|------|
| 화염 🔥 | 빙결 ❄️ | 화염 |
| 빙결 ❄️ | 번개 ⚡ | 빙결 |
| 번개 ⚡ | 대지 🌍 | 번개 |
| 대지 🌍 | 바람 💨 | 대지 |
| 바람 💨 | 화염 🔥 | 바람 |
| 신성 ✨ | 암흑 🌑 | 신성 |
| 암흑 🌑 | 신성 ✨ | 암흑 |

**속성 배율**:
- 약점: **1.5배** 데미지
- 저항: **0.5배** 데미지
- 무속성: **1.0배** (변동 없음)

## 🎭 상태 효과

### 버프 (긍정 효과)

| 이름 | 효과 | 지속시간 | 중첩 |
|------|------|----------|------|
| 공격력 증가 | 공격력 +20% | 3턴 | 최대 3 |
| 방어력 증가 | 방어력 +20% | 3턴 | 최대 3 |
| 헤이스트 | ATB +50% | 5턴 | 불가 |
| 리제네 | HP 회복 5%/턴 | 5턴 | 불가 |
| 프로텍트 | 물리 피해 -30% | 5턴 | 불가 |
| 쉘 | 마법 피해 -30% | 5턴 | 불가 |

### 디버프 (부정 효과)

| 이름 | 효과 | 지속시간 | 중첩 |
|------|------|----------|------|
| 공격력 감소 | 공격력 -30% | 3턴 | 최대 3 |
| 방어력 감소 | 방어력 -30% | 3턴 | 최대 3 |
| 슬로우 | ATB -50% | 5턴 | 불가 |
| 독 | HP 손실 3%/턴 | 5턴 | 최대 5 |
| 침묵 | 스킬 사용 불가 | 3턴 | 불가 |
| 스턴 | 행동 불가 | 1턴 | 불가 |
| 암흑 | 명중률 -50% | 3턴 | 불가 |

### 상처 시스템

**상처란?**
- HP 데미지의 25%가 상처로 누적
- 최대 누적: 최대 HP의 50%
- 자연 회복: 매 턴 최대 HP의 1%

**상처 효과**:
- 최대 HP 감소 (일시적)
- 회복 효율 저하

**상처 치료**:
1. **아이템 사용**: 회복 효율 1.5배
2. **휴식**: 상처 50% 회복
3. **시간 경과**: 자연 치유 (느림)

**상처 전략**:
- 상처가 많으면 회복 아이템 사용
- 긴 전투 전 휴식으로 상처 제거

## 🧠 전투 전략

### 기본 전략

**1. BRV 우선 축적**
```
초반 3턴: BRV 공격으로 BRV 1500 축적
4턴째: HP 공격으로 대량 데미지
```

**2. BREAK 노리기**
```
적 BRV 관찰 → 정확히 0으로 만들기 → 보너스 HP 공격
```

**3. 속성 활용**
```
적의 약점 확인 → 해당 속성 스킬 사용 → 1.5배 데미지
```

### 직업별 전략

**물리 딜러 (전사, 검성)**
```
1. BRV 공격 2회
2. HP 공격 1회
3. 궁극기 (BRV+HP 동시)
```

**마법 딜러 (아크메이지)**
```
1. 속성 약점 확인
2. 해당 속성 마법 (BRV 공격)
3. HP 공격
```

**탱커 (기사, 성기사)**
```
1. 방어 버프 사용
2. 적 어그로 유지
3. 아군 보호
```

**힐러 (성직자, 신관)**
```
1. 아군 HP 모니터링
2. HP < 50%일 때 힐
3. 버프 유지
```

### 고급 전략

**1. ATB 조작**
- 헤이스트 (아군): ATB +50%
- 슬로우 (적): ATB -50%
- 시간 차이로 2배 행동

**2. 상태 효과 조합**
```
공격력 증가 + 헤이스트 + 크리티컬 = 폭딜
방어력 감소 + 슬로우 = 적 무력화
```

**3. 파티 시너지**
```
1턴: 바드 (파티 버프)
2턴: 전사 (BRV 공격)
3턴: 아크메이지 (BRV 공격)
4턴: 전사 (BREAK + HP 공격)
```

**4. 자원 관리**
- MP 효율: 중요한 순간에만 스킬 사용
- BRV 관리: 너무 많이 쌓으면 손실 위험
- HP 관리: 상처 누적 주의

## 📊 난이도별 전투 팁

### 평온 (초보자)

- **전략**: 안전하게 BRV 충분히 쌓기
- **추천 직업**: 전사, 성직자
- **주의사항**: 여유있게 힐링

### 보통 (표준)

- **전략**: BREAK 노리기, 속성 활용
- **추천 직업**: 아크메이지, 암살자
- **주의사항**: MP 관리

### 도전 (숙련자)

- **전략**: 상태 효과 조합, 시너지
- **추천 직업**: 시간술사, 바드
- **주의사항**: 적 패턴 파악

### 악몽 (전문가)

- **전략**: 완벽한 ATB 조작, 최적화
- **추천 직업**: 차원술사, 해커
- **주의사항**: 실수 금지

### 지옥 (마스터)

- **전략**: 모든 시스템 완벽 활용
- **추천 직업**: 검성, 아크메이지
- **주의사항**: 단 한 번의 실수도 치명적

## 🎮 실전 예시

### 시나리오: 보스전 (고블린 왕)

**아군 파티**:
- 전사 (Lv.10, HP 210, BRV 1000)
- 성직자 (Lv.10, HP 150, BRV 800)

**적**:
- 고블린 왕 (Lv.10, HP 500, BRV 1200)

**전투 로그**:
```
[턴 1] 전사: BRV 공격 (강타)
  → 고블린 왕 BRV 1200 → 900 (-300)
  → 전사 BRV 1000 → 1300 (+300)

[턴 2] 성직자: 버프 (공격력 증가)
  → 전사 공격력 +20%

[턴 3] 고블린 왕: BRV 공격
  → 전사 BRV 1300 → 1100 (-200)
  → 고블린 왕 BRV 900 → 1100 (+200)

[턴 4] 전사: BRV 공격 (강타) [버프 적용]
  → 고블린 왕 BRV 1100 → 740 (-360)
  → 전사 BRV 1100 → 1460 (+360)

[턴 5] 성직자: BRV 공격 (신성 벌)
  → 고블린 왕 BRV 740 → 540 (-200)
  → 성직자 BRV 800 → 1000 (+200)

[턴 6] 전사: HP 공격 (십자 베기)
  → 고블린 왕 HP 500 → 134 (-366 HP)
  → 전사 BRV 1460 → 100 (초기화)

[턴 7] 고블린 왕: HP 공격 (분노의 일격)
  → 전사 HP 210 → 180 (-30 HP)
  → 고블린 왕 BRV 540 → 100

[턴 8] 성직자: 힐 (케어)
  → 전사 HP 180 → 210 (회복)

[턴 9] 전사: BRV 공격 (강타)
  → 고블린 왕 BRV 100 → 0 (BREAK!)
  → 전사 BRV 100 → 400

[턴 10] 전사: HP 공격 (BREAK 보너스)
  → 고블린 왕 HP 134 → 0 (사망!)
  → 승리!
```

# 🚀 Dawn of Stellar - 빠른 시작 가이드

## 일반 사용자를 위한 간단한 가이드

### ✅ 가장 쉬운 방법: 실행 파일 사용

1. **게임 다운로드**
   - 게임 파일이 들어있는 폴더를 다운로드하세요
   - 폴더 이름은 아무거나 상관없습니다

2. **게임 실행**
   - `DawnOfStellar.exe` 파일을 더블클릭하세요
   - 또는 `게임_실행.bat` 파일을 더블클릭하세요

3. **게임 시작!**
   - 게임이 자동으로 시작됩니다
   - Python이나 다른 프로그램을 설치할 필요가 없습니다!

---

## 문제가 발생했나요?

### 게임이 실행되지 않을 때

1. **Windows 보안 경고**
   - "Windows에서 이 앱을 실행할 수 없습니다" 메시지가 나오면:
   - 파일을 우클릭 → "속성" → "차단 해제" 체크 → 확인

2. **백신 프로그램 경고**
   - 일부 백신 프로그램이 실행 파일을 의심스러운 파일로 감지할 수 있습니다
   - 이는 정상적인 현상입니다 (게임 파일에 바이러스는 없습니다)
   - 필요시 예외 처리에 추가해주세요

3. **Visual C++ 오류**
   - "MSVCP140.dll을 찾을 수 없습니다" 같은 오류가 나오면:
   - 다음 링크에서 Visual C++ 재배포 가능 패키지를 설치하세요:
     https://aka.ms/vs/17/release/vc_redist.x64.exe

### 게임이 느릴 때

1. **다른 프로그램 종료**
   - 메모리 부족 시 게임이 느려질 수 있습니다

2. **백신 프로그램 실시간 검사 비활성화**
   - 게임 폴더를 예외 처리에 추가

### 한글이 깨질 때

1. **폰트 파일 확인**
   - 게임 폴더에 다음 파일이 있는지 확인:
     - `GalmuriMono9.bdf`
     - `dalmoori.ttf`
     - `DOSMyungjo.ttf`

2. **설정 파일 수정**
   - `config.yaml` 파일을 메모장으로 열기
   - `font_size: 60` 값을 `font_size: 32`로 변경해보기

## 📁 게임 폴더 구조

게임을 실행하면 다음 폴더가 자동으로 생성됩니다:

```
게임 폴더/
├── DawnOfStellar.exe      # 게임 실행 파일
├── 게임_실행.bat          # 간편 실행 스크립트
├── config.yaml            # 게임 설정
├── data/                  # 게임 데이터
├── assets/                # 오디오 파일
├── saves/                 # 저장 파일 (자동 생성)
└── logs/                  # 로그 파일 (자동 생성)
```

---

## 💾 저장 파일

- 게임 저장 파일은 `saves/` 폴더에 저장됩니다
- 다른 컴퓨터로 복사하여 사용할 수 있습니다
- 저장 파일은 JSON 형식입니다

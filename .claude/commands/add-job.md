# /add-job - 새 직업 추가 명령어

28개 직업 중 하나를 새 프로젝트에 추가합니다.

## 사용법
```
/add-job <job_name>
```

1. 전사, 2. 아크메이지, 3. 궁수, 4. 도적, 5. 성기사
6. 암흑기사, 7. 몽크, 8. 바드, 9. 네크로맨서, 10. 용기사
11. 검성, 12. 정령술사, 13. 암살자, 14. 기계공학자, 15. 무당
16. 해적, 17. 사무라이, 18. 드루이드, 19. 철학자, 20. 시간술사
21. 연금술사, 22. 검투사, 23. 기사, 24. 신관, 25. 마검사
26. 차원술사, 27. 광전사, 28. 마법사/성직자

## 예시
```bash
/add-job 전사
/add-job 아크메이지
```

---

**작업 내용:**

1. 기존 프로젝트에서 해당 직업의 데이터 추출
   - 기본 스탯
   - 고유 기믹 (예: 전사의 stance, 아크메이지의 element_cycle)
   - 특성(Trait) 5개
   - 전용 스킬 6개

2. 새 프로젝트에 추가
   - `data/characters/<job_name>.yaml` 생성
   - `src/character/classes/<job_name>.py` 생성
   - 테스트 파일 생성

3. 기믹 시스템 구현
   - 직업별 고유 메커니즘 구현
   - 이벤트 버스 통합

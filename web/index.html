⭐ Dawn of Stellar

별빛의 여명 - Final Fantasy 스타일 로그라이크 RPG

🎮 게임 소개

Dawn of Stellar는 Final Fantasy의 ATB 전투 시스템과 Dissidia의 BRV 시스템을 결합한 독창적인 로그라이크 RPG입니다.

✨ 핵심 특징

⚔️ 33개 고유 직업

각각 독자적인 기믹 시스템을 보유한 다양한 직업

🔮 ATB + BRV 전투

실시간 액션과 전략의 완벽한 조화

🎯 195개 이상 스킬

다채로운 전투 경험과 스킬 조합

🏰 절차적 던전 생성

매번 다른 도전과 탐험

🍳 요리 & 제작 시스템

52개 레시피와 서바이벌 요소

🎵 FF VII OST

오리지널 BGM과 효과음

🚀 시작하기

💻 로컬 플레이

git clone https://github.com/APTOL-7177/Dawn-of-Stellar.git
cd Dawn-of-Stellar
pip install -r requirements.txt
python main.py

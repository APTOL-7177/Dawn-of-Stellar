Dawn of Stellar - 다운로드 & 설치 가이드

시스템 요구사항

최소 사양
- 운영체제: Windows 10/11 (64비트)
- 프로세서: Intel Core i3 또는 동급 AMD 프로세서
- 메모리: 4GB RAM
- 저장공간: 500MB 여유 공간
- 그래픽: DirectX 11 지원 그래픽 카드
- 사운드: DirectX 호환 사운드 카드

권장 사양
- 운영체제: Windows 10/11 (64비트)
- 프로세서: Intel Core i5 또는 동급 AMD 프로세서
- 메모리: 8GB RAM 이상
- 저장공간: 1GB 여유 공간
- 그래픽: DirectX 11 지원 그래픽 카드 (1GB VRAM 이상)

다운로드

1. DawnOfStellar.zip 파일을 다운로드하세요
2. 파일 크기: 약 200MB (압축 해제 시 약 400MB)
3. 다운로드 완료 후 안전한 위치에 저장하세요

중요: 바이러스 백신 프로그램이 게임 파일을 차단할 수 있습니다.
게임 파일을 신뢰할 수 있는 경우 "허용" 또는 예외 처리하세요.

설치 및 실행

1단계: 압축 해제
DawnOfStellar.zip
├── DawnOfStellar.exe    # 메인 실행 파일 ⭐
├── _internal/           # 게임 엔진 파일들
├── config.yaml          # 게임 설정
├── assets/              # 게임 리소스
├── data/                # 게임 데이터
├── *.ttf, *.bdf         # 폰트 파일
└── user_data/           # 사용자 데이터 (자동 생성)

2단계: 실행
1. 압축 해제된 폴더에서 DawnOfStellar.exe 더블클릭
2. 게임이 자동으로 시작됩니다
3. 첫 실행 시 게임 데이터가 초기화됩니다

팁: 게임 실행 파일을 바탕 화면이나 시작 메뉴에 바로가기 아이콘으로 만들면 편리합니다!

게임 특징

완전 독립 실행
- Python 설치 불필요: 모든 라이브러리가 포함됨
- 추가 프로그램 없음: 별도의 설치 과정 없음
- 플러그 앤 플레이: 다운로드 후 즉시 실행 가능

게임 콘텐츠
- 33개 직업: 전사, 마법사, 궁수 등 다양한 캐릭터
- 336개 스킬: 풍부한 전투 시스템
- 완벽한 사운드: BGM과 효과음 지원
- 탐험 시스템: 마을과 던전 탐험
- 메타 진행: 별의 파편으로 콘텐츠 해금

사용자 친화적
- 자동 세이브: 게임 진행 상황 자동 저장
- 한글 지원: 완벽한 한국어 인터페이스
- 키보드 조작: 직관적인 컨트롤
- 고해상도 지원: 모니터 해상도 자동 조정

문제 해결

게임이 실행되지 않을 때
1. 안전한 위치 확인: 게임을 다운로드한 폴더에 실행 권한이 있는지 확인
2. 바이러스 백신: 게임 파일을 바이러스 백신 예외 목록에 추가
3. 관리자 권한: 필요한 경우 관리자 권한으로 실행
4. 호환성: Windows 10/11에서 실행 중인지 확인

그래픽 문제
- 게임이 검은 화면으로 나오면: 해상도 설정을 확인하세요
- 글자가 깨지면: 폰트 파일이 제대로 복사되었는지 확인하세요

사운드 문제
- 소리가 나지 않으면: 오디오 장치 설정을 확인하세요
- BGM이 끊기면: 다른 프로그램에서 오디오를 사용 중인지 확인하세요

저장 데이터 문제
- 게임이 시작되지 않으면: user_data 폴더를 삭제하고 다시 시작하세요
- 진행 상황이 사라지면: 백업본에서 복원하거나 새 게임을 시작하세요

성능 문제
- 게임이 느리면: 백그라운드 프로그램을 종료하세요
- 메모리 부족: 최소 4GB RAM 요구사항을 확인하세요

기술 지원

버그 리포트
게임 실행 중 문제가 발생하면 다음 정보를 포함해서 보고해주세요:
- 운영체제 버전
- 게임 버전 (제목 표시줄에 표시됨)
- 오류 메시지 (가능한 경우)
- 문제 발생 상황

로그 파일
문제가 발생하면 user_data/logs/ 폴더의 로그 파일을 확인해주세요.

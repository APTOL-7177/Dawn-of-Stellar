# Bot AI System - Archived

## 아카이브 이유
멀티플레이 호환성을 위해 봇 시스템을 제거했습니다.

## 아카이브된 파일
- `multiplayer/ai_bot.py` - 기본 AI 봇
- `multiplayer/ai_bot_advanced.py` - 고급 AI 봇
- `multiplayer/bot_communication.py` - 봇 통신 시스템
- `multiplayer/bot_tasks.py` - 봇 작업 시스템
- `ui/bot_help_ui.py` - 봇 도움말 UI
- `ui/bot_inventory_ui.py` - 봇 인벤토리 UI

## 복구 방법
필요 시 이 파일들을 다시 `src/` 폴더로 복사하여 사용할 수 있습니다.

```bash
# 복구 예시
copy archive\multiplayer\*.py src\multiplayer\
copy archive\ui\*.py src\ui\
```

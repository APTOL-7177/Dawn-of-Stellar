# Dawn of Stellar - 설치 가이드

## 📦 배포 버전 사용하기 (권장)

배포 버전은 Python을 설치하지 않아도 바로 실행할 수 있는 실행 파일입니다.

### 1. 게임 다운로드

1. 게임 파일이 들어있는 폴더를 다운로드하거나 복사합니다.
2. 폴더 이름은 `DawnOfStellar` 또는 원하는 이름으로 변경할 수 있습니다.

### 2. 게임 실행

#### Windows 사용자
- `DawnOfStellar.exe` 파일을 더블클릭하여 실행합니다.
- 또는 `게임_실행.bat` 파일을 실행합니다.

#### 문제 해결
- **"Windows에서 이 앱을 실행할 수 없습니다"** 오류가 나는 경우:
  - 파일을 우클릭 → "속성" → "차단 해제" 체크 → 확인
  - 또는 우클릭 → "관리자 권한으로 실행"

- **바이러스 백신 프로그램 경고**:
  - 일부 백신 프로그램이 실행 파일을 의심스러운 파일로 감지할 수 있습니다.
  - 이는 정상적인 현상이며, 게임 파일에 바이러스는 없습니다.
  - 필요시 예외 처리에 추가해주세요.

### 3. 게임 폴더 구조

```
DawnOfStellar/
├── DawnOfStellar.exe      # 게임 실행 파일
├── 게임_실행.bat           # 간편 실행 스크립트
├── config.yaml            # 게임 설정 파일
├── data/                   # 게임 데이터
├── assets/                 # 오디오 파일
├── saves/                  # 저장 파일 (자동 생성)
└── logs/                   # 로그 파일 (자동 생성)
```

## 🔧 개발자용: 직접 빌드하기

게임을 직접 빌드하고 싶은 경우 다음 단계를 따르세요.

### 필요 사항

- Python 3.10 이상
- 인터넷 연결 (패키지 다운로드용)

### 빌드 방법

1. **Python 설치 확인**
   ```bash
   python --version
   ```
   Python 3.10 이상이 설치되어 있어야 합니다.

2. **의존성 설치**
   ```bash
   pip install -r requirements.txt
   pip install pyinstaller
   ```

3. **빌드 실행**
   - Windows: `build_executable.bat` 파일을 더블클릭
   - 또는 명령줄에서:
     ```bash
     python -m PyInstaller build.spec --clean --noconfirm
     ```

4. **빌드 완료**
   - 빌드가 완료되면 `dist` 폴더에 `DawnOfStellar.exe` 파일이 생성됩니다.
   - `dist` 폴더의 모든 내용을 다른 컴퓨터로 복사하면 Python 없이도 실행할 수 있습니다.

### 빌드 옵션

`build.spec` 파일을 수정하여 빌드 옵션을 변경할 수 있습니다:

- **콘솔 창 숨기기**: `console=False`로 변경
- **아이콘 추가**: `icon="경로/아이콘.ico"` 추가
- **UPX 압축 비활성화**: `upx=False`로 변경 (일부 백신에서 오탐지 방지)

## 🎮 게임 실행 방법

### 기본 실행
1. `DawnOfStellar.exe` 파일을 더블클릭
2. 게임이 시작됩니다!

### 명령줄 옵션 (개발자용)

명령줄에서 실행할 때 다음 옵션을 사용할 수 있습니다:

```bash
DawnOfStellar.exe --dev          # 개발 모드 (모든 클래스 잠금 해제)
DawnOfStellar.exe --debug        # 디버그 모드
DawnOfStellar.exe --log=DEBUG    # 상세 로그 출력
```

게임 저장 파일은 다음 위치에 저장됩니다:

## 🐛 문제 해결

### 게임이 실행되지 않는 경우

1. **Windows Defender 또는 백신 프로그램 확인**
   - 실행 파일이 차단되었을 수 있습니다.
   - 예외 처리에 추가해주세요.

2. **Visual C++ 재배포 가능 패키지 설치**
   - 일부 시스템에서 필요할 수 있습니다.
   - Microsoft 공식 사이트에서 다운로드:
     https://aka.ms/vs/17/release/vc_redist.x64.exe

3. **로그 파일 확인**
   - `logs/` 폴더의 로그 파일을 확인하여 오류 원인을 파악할 수 있습니다.

### 게임이 느리게 실행되는 경우

1. **백신 프로그램 실시간 검사 비활성화**
   - 게임 폴더를 예외 처리에 추가

2. **다른 프로그램 종료**
   - 메모리 부족 시 게임이 느려질 수 있습니다.

### 한글이 깨지는 경우

1. **폰트 파일 확인**
   - 게임 폴더에 다음 폰트 파일이 있는지 확인:
     - `GalmuriMono9.bdf`
     - `dalmoori.ttf`
     - `DOSMyungjo.ttf`

2. **설정 파일 확인**
   - `config.yaml` 파일의 `display.font_size` 값을 조정해보세요.
